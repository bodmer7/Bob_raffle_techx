Monospace chip for any number the game surfaces — drop odds, owned counts, prices, quantities. Tabular figures keep columns aligned.

```jsx
<StatPill label="Odds" value="0.64%" tone="rarity" />
<StatPill label="Owned" value="x3" />
<StatPill icon={<span className="bob-codemark">{'</>'}</span>} value="1,280" tone="accent" />
```

Tones: `neutral | accent | success | rarity` (rarity reads `[data-rarity]`). Sizes `sm | md`.
