The case the player opens — a glossy IBM-blue capsule on a glowing pedestal, stamped with the `</>` mark, idly floating with a shine sweep. Pair it with an `exotic` Button to trigger the open.

```jsx
<Capsule name="Legendary Capsule" series="Series 02" hint="legendary" />
<Button variant="exotic" size="lg">Open · 1 Key</Button>
```

`hint` tints the aura + the glow through the glass to tease what's inside. Set `floating={false}` for static thumbnails.
